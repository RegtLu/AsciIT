### 配置要求
- [FFmpeg](https://www.ffmpeg.org/)
- [Python](https://www.python.org/)
  - Pillow
  - tqdm
  - playsound==1.2.2
- [sarasa-term-sc-nerd-regular.ttf](https://github.com/laishulu/Sarasa-Term-SC-Nerd)(可替换)

### 快速开始
>`start.py video.mp4`
>生成video.mp4的字符版本
